# Notification Sounds

This directory contains audio files for the notification system.

## Required Files

- `notification.mp3` - Default notification sound

## Adding Sounds

Place your notification audio files in this directory. Recommended formats:
- MP3 (best compatibility)
- OGG (open format)
- WAV (uncompressed)

## Sound Guidelines

- Duration: 0.5-2 seconds
- Volume: Not too loud
- Frequency: Pleasant, non-intrusive
- Format: Mono or stereo, 44.1kHz or 48kHz

## Example Sources

- Free sound libraries (Freesound.org, Zapsplat.com)
- Custom synthesized sounds
- System notification sounds (check licensing)
